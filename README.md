🚗 Carola | Araç Kiralama Sistemi

Carola, araç kiralama operasyonlarını dijital ortamda daha hızlı, düzenli ve kullanıcı dostu şekilde yönetmek amacıyla geliştirilen bir web uygulamasıdır. Proje N-Tier Architecture yapısına uygun olarak geliştirilmiştir.

⚙️ Temel Özellikler
Araç listeleme ve filtreleme
Yakıt tipi, vites türü ve fiyat aralığına göre filtreleme
Rezervasyon yönetimi
Adım adım rezervasyon akışı ve fiyat özeti
Admin paneli
Araç, rezervasyon, kullanıcı, kategori ve lokasyon yönetimi
MailKit / SMTP entegrasyonu ile rezervasyon onaylandığında müşteriye rezervasyon özeti ve tek kullanımlık kupon kodu iletilmesi

🛠️ Kullanılan Teknolojiler
ASP.NET Core 6 MVC
Entity Framework Core
AutoMapper
FluentValidation
MailKit (SMTP)
HTML / CSS / Bootstrap
